Izin GPS Diblokir

Cara mengaktifkan akses lokasi aplikasi

Aplikasi membutuhkan akses GPS untuk memverifikasi posisi Anda secara presisi. Silakan pilih jenis perangkat Anda di bawah untuk melihat panduan:

1

Ketuk ikon gembok / pengaturan situs di sebelah kiri alamat website (URL) di bagian atas browser Anda.

2

Pilih menu Izin (Permissions) atau Setelan Situs (Site Settings).

3

Aktifkan atau ubah status Lokasi (Location) menjadi Izinkan (Allow).

4

Refresh / muat ulang halaman ini.

Ketuk ikon aA di sebelah kiri kolom alamat URL Safari.

2

Pilih Pengaturan Situs Web (Website Settings).

3

Ubah opsi Lokasi (Location) menjadi Izinkan (Allow).

Masih tidak bisa?

Buka Pengaturan iOS > Privasi & Keamanan > Layanan Lokasi. Pastikan Layanan Lokasi aktif, lalu scroll ke bawah pilih Situs Web Safari, ubah ke Saat App Digunakan.

Klik ikon gembok yang ada di sebelah kiri alamat URL website pada browser Anda.

3
